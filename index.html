Angular JS

How to use the module in your own Angular application

  1. Include the CSS: 
    <script src="//ajax.googleapis.com/ajax/libs/angularjs/1.0.7/angular.min.js"></script>
  2. Include the JS: 
    <script src="ng-cast-receiver-list.js"></script>
  3. Configure the module with your Application ID: 
    <script>angular.module('crl').constant('CAST_APP_ID', 'xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx');</script>
  4. Register the module as a dependency for your Angular module: 
    <script>angular.module('app', [ 'crl' ]);</script>
